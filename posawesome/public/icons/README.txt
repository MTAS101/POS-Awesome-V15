# PWA Icons for POS Awesome

For proper PWA functionality, please create and place the following icon files in this directory:

- icon-72x72.png (72x72 pixels)
- icon-96x96.png (96x96 pixels) 
- icon-128x128.png (128x128 pixels)
- icon-144x144.png (144x144 pixels)
- icon-152x152.png (152x152 pixels)
- icon-192x192.png (192x192 pixels)
- icon-384x384.png (384x384 pixels)
- icon-512x512.png (512x512 pixels)

These icons are referenced in the manifest.json file and are required for proper installation as a PWA on various devices.

Use your company logo or the POS Awesome logo, ensuring proper scaling for each size.
